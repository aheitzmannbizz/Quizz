Quiz app built with Streamlit.

Local run
- Activate environment and run: python -m streamlit run app.py
- Or use: .\start_ios_local.ps1

Publish on Streamlit Community Cloud
1. Push this project to a public GitHub repository.
2. Open https://share.streamlit.io and sign in with GitHub.
3. Click New app.
4. Select your repository and branch.
5. Set main file path to app.py.
6. Click Deploy.

Notes
- Dependencies are read from requirements.txt.
- Do not commit secrets. If you later need secrets, add them in Streamlit Cloud app settings.
